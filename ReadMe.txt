** Logistic Regression model with User Interface **

Install anaconda Navigator & use jupyter notebook.

Required tools, languages & packages

* Language: Python 2 or above.
* Pandas For Dataframe: install pandas
* Mathematical operations: install numpy
* Machine Learning algorithms: install scikitlearn
* Data Visualisation: install matplotlib

For flask install flask in commmand prompt using python 

Files & Folders
* models - Consist of the model used for prediction
* static - Consists of images, java scripts and css
* templates	-index_1.html : User Input
		-results_1.html: User Output 

Data File: logistic_regression_updated_csv 
Classified_Supertar.csv - original Raw data

Python File: app.py: Model & API Flask is used	

Run the program
* Open command prompt
* Change directory to the output file
* Type app.py and press Enter
* If you have python more than version 2, the code will run
* If it runs, a local host link will be presented
* Copy the link and paste on chrome.
* Type the values and click 'Predict your potential'.
